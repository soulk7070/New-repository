# ü§ñ Setup Bot Telegram untuk Workflow

## Langkah 1: Aktifkan Bot dan Dapatkan Chat ID

1. **Start Bot Anda**
   - Buka bot Telegram Anda
   - Ketik `/start`
   - Bot akan aktif

2. **Dapatkan Chat ID** (untuk output notification)
   - Kirim pesan apapun ke bot Anda
   - Buka browser, buka URL ini (ganti TOKEN dengan bot token Anda):
   ```
   https://api.telegram.org/botYOUR_BOT_TOKEN/getUpdates
   ```
   - Cari bagian `"chat":{"id":` - angka setelahnya adalah Chat ID Anda
   - Catat Chat ID ini (misal: 123456789)

## Langkah 2: Test Bot Token

Untuk memastikan bot token bekerja, test dengan URL ini:
```
https://api.telegram.org/botYOUR_BOT_TOKEN/getMe
```

Jika berhasil, akan muncul info bot dalam format JSON.
